# 莱洛三角形  
## 一.项目简介
莱洛三角形是一位大佬开源的项目，我趁着寒假时间复刻了出来。目前软件还没有自己写，烧录的是开源的程序，本项目主要学习的点在于无刷电机FOC、控制算法和上位机开发。
